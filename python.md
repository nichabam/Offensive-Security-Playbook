# Python notes

## Virtual Environment

```
python3 -m venv venv

# activate
source venv/bin/activate

# Installing packages
pip install <package>

pip install -r requirements.txt

# Deactivate
deactivate

# If venv is missing
sudo apt install python3-venv
```
